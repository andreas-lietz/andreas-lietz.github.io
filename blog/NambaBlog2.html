The Basics of Namba Forcing II

An attempt at more minimal proofs. ???, 2024

In a prior post we tried to give a concise proof of some basic facts about Namba forcing, culminating in a proof that Namba forcing preserves stationary subsets of $\omega_1$.

Here, I will continue my quest to find and present more simple proofs for nice facts about Namba forcing. We will proof a result due to Jensen: If $G$ is $\mathsf{Nm}$-generic and $\vec\alpha=\langle\alpha_n\mid n<\omega\rangle\in V[G]$ is cofinal in $\omega_2^V$ then, considered as a branch through $(\omega_2^V)^{<\omega}$, $\vec \alpha$ is a $\mathsf{Nm}$-generic branch over $V$. Moreover, $V[\vec\alpha]=V[G]$.

Assuming $\mathrm{CH}$, this is also a consequence of a general result due to Bukovsky.

Recall that a Namba tree is a subtree $p\subseteq\omega_2^{<\omega}$ so that for each $t\in p$, $\{s\in p\mid t\subseteq s\}$ has size $\omega_2$. There is another version of Namba forcing: A Namba' tree is a subtree $p\subseteq \omega_2^{<\omega}$ so that every node above the stem of $p$ has $\omega_2$-many immediate successors (i.e. is a splitting node).

Every Namba' tree is a Namba tree but not vice versa. $\mathsf{Nm}, \mathsf{Nm}'$ are the collection of Namba, Namba' trees resprectively ordered by inclusion. Both $\mathsf{Nm}, \mathsf{Nm}'$ have the same basic properties covered in the first blog post. However, Magidor-Shelah have shown that under $\mathrm{CH}$, $\mathsf{Nm}$ and $\mathsf{Nm}'$ are not forcing equivalent in a strong sense: A $\mathsf{Nm}$-generic extension does not contain a $\mathsf{Nm}'$-generic and vice versa. We will improve their result here by dropping the $\mathrm{CH}$ assumption.

Theorem

(Jensen) Suppose that $G$ is $\mathsf{Nm}$-generic and $c\in V[G]$ is a branch through $(\omega_2^V)^{<\omega}$ so that $\mathrm{ran}(c)$ is cofinal in $\omega_2^V$. Then $c$ is a $\mathsf{Nm}$-generic branch over $V$ and $V[c]=V[G]$.

Proof

We will show that $H_c=\{q\in\mathsf{Nm}\mid c\in[q]\}$ is a $V$-generic filter with $G\in V[H_c]$.

We leave it to the reader to show that $H_c$ is a filter and only show the genericity part. So let us work in $V$ and let $\dot c, \dot H_c$ be a $\mathsf{Nm}$-names for $c, \dot H_c$ and let $p\in\mathsf{Nm}$ force the relevant properties of $\dot c$.

We will find some $q\leq p$ with the following property:

If $s, t\in q$ are incompatible then $s, t$ disagree about $\dot c$. More precisely, there are incompatible $u,v\in\omega_2^{<\omega}$ so that both $q\upharpoonright s\Vdash\check u\subseteq\dot c$ and $q\upharpoonright t\Vdash \check v\in\dot c$.

We build $q\leq p$ via a fusion argument. simultaneously, we define a partial map $\mu\colon \omega_2^{<\omega}\rightarrow\omega_2^{<\omega}$. So let us construct a fusion sequence $(p_n)_{n<\omega}$ starting with $p_0=p$ as follows:

Suppose $p_n$ is already defined and $t\in p_n$ is a $n$-splitting node of $p_n$. We describe how to define $p_{n+1}\upharpoonright t$. Let $$\langle s_i\mid i<\omega_2\rangle$$ enumerate all immediate successors of $t$ in $p_n$. By induction on $i<\omega_2$, we define a sequences $\langle r_i\mid i<\omega_2\rangle$ and $\langle u_i\mid i<\omega_2\rangle$ so that

$(1)$ $r_i\leq p_n\upharpoonright s_i$, $u_i\in \omega_2^{<\omega}$,

$(2)$ $r_i\Vdash \check u_i\subseteq \dot c$ and

$(3)$ $u_i$ is different from every $u_j$, $j\neq i$ (but not necessarily incompatible with all those).

The construction is simple: if these objects are defined for $j < i$ then we pick any $r_i, u_i$ so that $(1), (2)$ hold and $u_i$ is different from all $u_j$ with $j < i$. Such $r_i$, $u_i$ must exist, otherwise $p_n\upharpoonright s_i\Vdash\dot c\subseteq \{u_i\mid i<\omega_2\}^{\check{}}$. But then $p_n\upharpoonright s_i$ forces $\mathrm{ran}(\dot c)$ to be bounded below $\omega_2$, contradiction.

Next, there must be some $m < \omega$ so that $$X=\{i<\omega_2\mid \mathrm{len}(u_i)=m\}$$ is of size $\omega_2$. It follows that $u_i, u_j$ are incompatible if $i\neq j$ are both in $X$. We then define $$p_{n+1}\upharpoonright t = \bigcup_{i\in X} r_i.$$ Also, for $i\in X$ we put $s_i$ into the domain of $\mu$ and let $\mu(s_i)=u_i$.

This concludes the construction of the fusion sequence and of $\mu$. We then let $q$ be stronger than the fusion along $(p_n)_{n<\omega}$ so that every node of $q$ either splits or has a unique successor. It should be clear that $q$ has the desired property.

For convenienve, we extend $\mu$ to all of $q$. Note that $\mu$ is defined for each immediate successor of a splitting node in $q$. For $t$ an initial segment of the stem of $q$, let $\mu(t)=\emptyset$. For other $t\in q$ let $s\leq_q t$ be the largest initial segment of $t$ which is an immediate successor of a splitting node of $q$ and set $\mu(t)=\mu(s)$.

The map $\mu$ has the following convenient properties:

$(1)$ If $s, t\in q$ are incompatible then $\mu(s),\mu(t)$ are incompatible as well.

$(2)$ For $t\in q$, we have $q\upharpoonright t\Vdash \mu(t)\subseteq\dot c$.

It follows from $(1)$ that the tree $$r=\{s\in\omega_2^{<\omega}\mid\exists t\in q\ s\subseteq \mu(t)\}$$ is a Namba tree and from $(2)$ that $q\Vdash\check r\in \dot H_c$.

Claim

There is an isomorphism $\pi\colon \mathsf{Nm}\upharpoonright r\rightarrow \mathsf{Nm}\upharpoonright q$ so that $\pi(r')\Vdash \check r'\in\dot H_c$ for all $r'\leq r$.

Proof

Define $\pi$ via $\pi(r')=\mu^{-1}[r']$. Note that any $s\in r'$ can be extended to some $s\in r'$ which is in the range of $\mu$: find a maximal initial segment of $t$ of the form $\mu(u)$. Then by construction of $q$, there is some $m<\omega$ so that all extension of $\mu(u)$ in $r$ of length $m$ are in the range of $\mu$ and hence $t$ is below one of those. This easily implies that $\pi(r')\in\mathsf{Nm}$ and that $\pi$ is injective. The remaining properties we ask of $\pi$ are straightforward to check.

$\Box$

It follows that if $G$ is $\mathsf{Nm}$-generic with $q\in G$ and $c=\dot c^G$ then $H_c$ is the upwards closure of $\pi[\{q'\in G\mid q\leq q\}]$. In particular, $H_c$ is $\mathsf{Nm}$-generic over $V$. Similarly, $G$ is the upwards closure of $\pi^{-1}[\{r'\in H_c\mid r\leq r\}]$ so $G\in V[H_c]$ and consequently $V[G]=V[c]$.

$\Box$

A similar argument shows that the theorem above is valid for $\mathsf{Nm}'$ as well.

Corollary

If $G$ is $\mathsf{Nm}$-generic then there is no $\mathsf{Nm}'$-filter generic over $V$ in $V[G]$ and vice versa. In particular, $\mathsf{Nm}$, $\mathsf{Nm}'$ are not forcing equivalent and neither forcing embeds regularly into the other.

Proof

This follows from Jensen's theorem above and the following observation: If $G$ is $\mathsf{Nm}$-generic and $b$ is the resulting generic branch then there is a function $F\colon(\omega_2^V)^{<\omega}\rightarrow(\omega_2^V)^{<\omega}$ so that

• $F\in V$ and

• there are unboundedly many $n<\omega$ so that $F(b\upharpoonright n) = b\upharpoonright (n+1)$.

So in essence, $F\in V$ is somewhat successful in predicting the generic branch. This is seen as follows: simply find some $p\in G$ so that any $t\in p$ can be extended to some $s\in p$ which has a unique immediate successor in $p$. Such conditions form a dense open subset of $\mathsf{Nm}$ so such $p$ exists in $G$. We then define $F(u)$ as the immediate successor of $u$ in $p$ if both $u\in p$ and $u$ has such a unique immediate successor in $p$. Otherwise, define $F(u)$ arbitrarily. It should be clear that $p$ forces $F$ to witness the above.

On the other hand, for a branch $b'$ generic for $\mathsf{Nm}'$ over $V$, such an $F$ similarly predicting $b'$ cannot exist in $V$ by a simple density argument. But if $c$ existed in a $\mathsf{Nm}$-extension of $V$ then $c$ would be $\mathsf{Nm}$-generic over $V$ as well, so such an $F$ exists nontheless, contradiction. Vice versa, a symmetric argument shows that $b$ cannot exist in a $\mathsf{Nm}'$-generic extension of $V$ either.

$\Box$

TL;DR

We give a simplified proof of a result due to Jensen and use it to improve a theorem of Magidor-Shelah by dropping their $\mathrm{CH}$ assumption.
